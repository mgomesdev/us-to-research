## UC07 - Finalizar Checkout com Pix

- **Caso de Uso Geral:** Checkout
- **Ator Principal:** Participante
- **Atores Secundários:** Sistema Financeiro
- **Resumo:** Este caso de uso representa o processo de pagamento via Pix. O sistema gera QR Code e código Pix copia e cola, e aguarda confirmação do pagamento via webhook.
- **Pré-condições:** Dados do comprador preenchidos e Pix selecionado
- **Pós-condições:** Pagamento confirmado ou expirado

### Cenário Principal

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| 1. Seleciona "Pix" | 2. Exibe QR Code e código Pix copia e cola |
| | 3. Exibe tempo restante para pagamento |
| 4. Realiza pagamento no app bancário | 5. Aguarda confirmação via webhook |
| | 6. Pagamento confirmado: exibe tela de confirmação |

### Cenário Alternativo

**Expiração do tempo:**

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| | 1. Tempo expira: redireciona para página do evento |
| 2. Reinicia checkout se ingressos disponíveis | |

### Cenário de Exceção

- Pagamento não confirmado após timeout: redirecionar para página do evento

### Requisitos Não-Funcionais

- Exibir tempo regressivo claramente
- QR Code deve ser gerado em tempo real
