## UC05 - Aplicar Cupom de Desconto

- **Caso de Uso Geral:** Checkout
- **Ator Principal:** Participante
- **Atores Secundários:** -
- **Resumo:** Este caso de uso representa a aplicação de cupom de desconto no painel de resumo do pedido. O participante informa o código do cupom e o sistema valida sua aplicabilidade.
- **Pré-condições:** Ingressos selecionados
- **Pós-condições:** Cupom aplicado ou rejeitado com mensagem

### Cenário Principal

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| 1. Acessa painel de resumo do pedido | 2. Exibe campo de cupom com botão "Adicionar" |
| 3. Informa código do cupom | |
| 4. Clica em "Adicionar" | 5. Valida cupom |
| | 6. Exibe desconto no resumo com nome e valor |
| | 7. Exibe opção para remover cupom |

### Cenário Alternativo

**Cupom inválido:**

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| | 1. Exibe mensagem de erro abaixo do campo |

### Restrições/Validações

- Apenas um cupom por pedido
- Sem restrição por CPF ou e-mail
- Validar: cupom válido, não expirado, encontrado

### Cenário de Exceção

- Cupom expirado: mensagem "Cupom expirado"
- Cupom não encontrado: mensagem "Cupom não encontrado"
- Cupom inválido: mensagem "Cupom inválido"
