## UC09 - Finalizar Checkout Gratuito

- **Caso de Uso Geral:** Checkout
- **Ator Principal:** Participante
- **Atores Secundários:** -
- **Resumo:** Este caso de uso representa o processo de finalização de checkout para eventos ou ingressos gratuitos. Não há método de pagamento, apenas confirmação da inscrição.
- **Pré-condições:** Ingressos gratuitos selecionados
- **Pós-condições:** Inscrição confirmada

### Cenário Principal

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| 1. Seleciona ingresso gratuito | 2. Exibe Bloco 1 (se nominal) e Bloco 2 |
| 3. Preenche dados necessários | 4. Valida dados |
| | 5. Não exibe Bloco 3 com métodos de pagamento |
| | 6. Exibe botão de confirmação da inscrição |
| 7. Clica em "Confirmar inscrição" | 8. Procesa inscrição |
| | 9. Exibe tela de confirmação |

### Cenário Alternativo

**Checkout não nominal gratuito:**

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| | 1. Exige autenticação antes de acessar checkout |
| 2. Não está logado: redireciona para login/cadastro | |
| 3. Realiza login/cadastro | 4. Retorna ao checkout |
| | 5. Não exibe Bloco 1 |
| | 6. Exibe Bloco 2 |
| | 7. Não exibe Bloco 3 |

### Restrições/Validações

- Checkout não nominal gratuito exige autenticação obrigatória

### Cenário de Exceção

- Usuário não autenticado tentando avançar: redirecionar para login
