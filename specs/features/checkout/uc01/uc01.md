## UC01 - Preencher Dados dos Participantes

- **Caso de Uso Geral:** Checkout
- **Ator Principal:** Participante
- **Atores Secundários:** -
- **Resumo:** Este caso de uso representa o processo por meio do qual o participante preenche os dados de cada ingresso adquirido no checkout nominal. Cada ingresso possui um card de preenchimento contendo campos obrigatórios (nome, e-mail, celular) e campos customizados configurados pelo organizador. O sistema valida os dados antes de permitir o avanço para o próximo bloco.
- **Pré-condições:** Participante ter selecionados ingressos nominais no evento
- **Pós-condições:** Dados dos participantes validados e prontos para próximos passos

### Cenário Principal

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| 1. Acessa o checkout com ingressos selecionados | 2. Exibe o Bloco 1 - Dados dos Participantes com cards para cada ingresso |
| 3. Preenche os campos obrigatórios de cada card (nome, e-mail, celular) | 4. Valida formato do e-mail e telefone em tempo real |
| | 5. Exibe campos adicionais configurados pelo organizador (CPF, empresa, cargo, etc.) |
| 6. Preenche campos adicionais quando obrigatórios | 7. Valida campos obrigatórios |
| | 8. Exibe indicação visual de campos válidos |
| 9. Clica em "Avançar" | 10. Valida todos os campos obrigatórios |
| | 11. Permite avanço se todos os campos estiverem válidos |

### Restrições/Validações

- Campo nome: obrigatório, texto livre
- Campo e-mail: obrigatório, formato válido de e-mail
- Campo celular: obrigatório, com seletor de DDI (Brasil pré-selecionado)
- Campos customizados: seguem obrigatoriedade configurada pelo organizador
- Sistema deve validar antes de permitir avanço

### Cenário Alternativo

**Checkout não nominal:** Bloco 1 não é exibido

### Cenário de Exceção

- Campo obrigatório não preenchido: exibir indicação visual e mensagem explicativa
- Dados inválidos: exibir mensagem de erro específica por campo

### Requisitos Não-Funcionais

- Validação em tempo real para melhor experiência do usuário
- Máscara automática para telefone
