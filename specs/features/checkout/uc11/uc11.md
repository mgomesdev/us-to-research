## UC11 - Exibir Tela de Confirmação

- **Caso de Uso Geral:** Checkout
- **Ator Principal:** Participante
- **Atores Secundários:** -
- **Resumo:** Este caso de uso representa a exibição da tela de confirmação de compra/inscrição após conclusão bem-sucedida do checkout.
- **Pré-condições:** Checkout concluído com sucesso
- **Pós-condições:** Tela de confirmação exibida

### Cenário Principal

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| | 1. Checkout aprovado |
| | 2. Exibe tela de confirmação (mesma já existente) |

### Cenário Alternativo

**Após webhook de pagamento confirmado (Pix/Boleto):**

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| | 1. Recebe confirmação de pagamento |
| | 2. Envia e-mail com ingressos |
| | 3. Exibe tela de confirmação |

### Requisitos Não-Funcionais

- Manter comportamento da tela de confirmação existente
