## UC03 - Selecionar Método de Pagamento

- **Caso de Uso Geral:** Checkout
- **Ator Principal:** Participante
- **Atores Secundários:** -
- **Resumo:** Este caso de uso representa a seleção do método de pagamento entre as opções disponíveis para o evento (Pix, Cartão de crédito, Boleto). Apenas os métodos habilitados pelo organizador são exibidos.
- **Pré-condições:** Bloco 2 concluído
- **Pós-condições:** Método de pagamento selecionado

### Cenário Principal

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| 1. Acessa o Bloco 3 - Pagamento | 2. Exibe opções de pagamento disponíveis (Pix, Cartão, Boleto) |
| 3. Seleciona um método de pagamento | 4. Marca método selecionado |
| 5. Clica em "Avançar para pagamento" | 6. Valida seleção e prossegue |

### Restrições/Validas/Validações

- Apenas métodos habilitados pelo organizador são exibidos
- Participante deve selecionar um método para prosseguir

### Cenário Alternativo

**Checkout gratuito:** Bloco 3 não exibe métodos de pagamento

### Requisitos Não-Funcionais

- Interface clara com ícones para cada método
