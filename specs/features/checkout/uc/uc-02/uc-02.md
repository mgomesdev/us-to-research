## UC02 - Selecionar responsável pelos ingressos

- **Ator Principal:** Participante
- **Atores Secundários:**
- **Resumo:** Este caso de uso representa o processo por meio do qual o participante seleciona quem será o responsável pelos ingressos adquiridos. O sistema exibe um dropdown com os participantes informados no Bloco 1 mais a opção "Outro". O responsável receberá os ingressos por e-mail e terá uma conta criada automaticamente na plataforma.
- **Pré-condições:** Dados dos participantes preenchidos (para checkout nominal) ou participante logado (para checkout não nominal).
- **Pós-condições:** Responsável pelosingressos definido e vinculado ao processo de checkout.

### Cenário Principal

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| 1. Visualiza dropdown "Responsável pelos ingressos" | 2. Exibe opções: participantes do Bloco 1 + "Outro" |
| 3. Seleciona uma das opções | 4. Se participante: usa e-mail automaticamente |
| | 5. Se "Outro": exibe campos para preenchimento |
| 6. Preenche nome completo (se "Outro") | 7. Valida campo obrigatório |
| 8. Preenche e-mail (se "Outro") | 9. Valida formato de e-mail |
| 10. Preenche celular (se "Outro") | 11. Valida formato de celular |
| 12. Marca checkbox "Eu li e concordo com os termos e condições" | 13. Valida checkbox marcado |
| 14. (Opcional) Marca checkbox "Aceito receber contato..." | |
| 15. Clica em botão para avançar | 16. Exibe nota informativa sobre criação automática de conta |

### Restrições/Validações

- Dropdown "Responsável pelosingressos" só é exibido no checkout nominal
- No checkout não nominal, este bloco não é exibido
- Campos de "Outro" são obrigatórios quando selecionados
- Checkbox de termos e condições é obrigatório para avanço
- Checkbox de recebimento de contato é opcional

### Cenário Alternativo

- **Checkout não nominal pago:** Bloco 2 é exibido sem dropdown. Usuário preenche dados manualmente.
- **Checkout não nominal gratuito:** Exige autenticação prévia.

### Cenário de Exceção

- E-mail informado já possui conta: sistema usa conta existente automaticamente
- Campos obrigatórios não preenchidos: sistema exibe mensagem de erro
