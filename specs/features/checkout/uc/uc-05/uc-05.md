## UC05 - Aplicar cupom de desconto

- **Ator Principal:** Participante
- **Atores Secundários:**
- **Resumo:** Este caso de uso representa o processo por meio do qual o participante pode aplicar um cupom de desconto para obter redução no valor total da compra. O campo está disponível no painel de resumo do pedido.
- **Pré-condições:** Ingressos selecionados e checkout em andamento.
- **Pós-condições:** Cupom aplicado com desconto refletido no valor total, ou erro informado.

### Cenário Principal

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| 1. Localiza campo de cupom no painel de resumo | 2. Exibe campo de texto e botão "Adicionar" |
| 3. Digita código do cupom | 4. Habilita botão "Adicionar" |
| 5. Clica em "Adicionar" | 6. Valida cupom |
| | 7. Se válido: aplica desconto e exibe no resumo |
| | 8. Se inválido: exibe mensagem de erro |

### Restrições/Validações

- Apenas um cupom por pedido
- Não há regras de restrição por CPF ou e-mail
- Cupom pode ser removido após aplicado

### Cenário Alternativo

- **Cupom válido:** Sistema exibe nome do cupom e valor descontado em destaque, com opção de remover.
- **Cupom inválido/expirado/não encontrado:** Sistema exibe mensagem de erro clara abaixo do campo.

### Cenário de Exceção

- Cupom expirado durante checkout: sistema mantém cupom até finalização ou remove com aviso
