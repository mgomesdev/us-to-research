## UC03 - Selecionar método de pagamento

- **Ator Principal:** Participante
- **Atores Secundários:** Sistema de Pagamento
- **Resumo:** Este caso de uso representa o processo por meio do qual o participante seleciona o método de pagamento desejado para finalizar a compra. O sistema exibe as opções habilitadas pelo organizador: Pix, Cartão de crédito ou Boleto.
- **Pré-condições:** Blocos 1 e 2 concluídos corretamente.
- **Pós-condições:** Método de pagamento selecionado e preparar interface para coleta de dados específicos do método.

### Cenário Principal

| Ações do Ator | Ações do Sistema |
|---------------|------------------|
| 1. Visualiza opções de método de pagamento | 2. Exibe apenas métodos habilitados pelo organizador |
| 3. Seleciona método de pagamento | 4. Destaca método selecionado |
| 5. Confirma seleção | 6. Exibe campos específicos do método selecionado |

### Restrições/Validações

- Apenas métodos habilitados pelo organizador são exibidos
- Participante deve selecionar um método para prosseguir
- Métodos disponíveis: Pix, Cartão de crédito, Boleto

### Cenário Alternativo

- **Checkout gratuito:** Bloco 3 não exibe métodos de pagamento. Sistema exibe apenas botão de confirmação.

### Cenário de Exceção

- Nenhum método disponível: sistema exibe mensagem de erro indicando indisponibilidade
